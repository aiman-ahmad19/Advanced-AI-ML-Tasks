# News Topic Classifier Using BERT

## Objective
Fine-tune a transformer model (BERT) to classify news headlines into topic categories using the AG News Dataset.

## Methodology / Approach
1. **Data Loading**: Load the AG News dataset from Hugging Face Datasets
2. **Preprocessing**: Tokenize the text using BERT tokenizer
3. **Model Development**: Fine-tune the bert-base-uncased model for text classification
4. **Training**: Train the model with appropriate hyperparameters
5. **Evaluation**: Evaluate using accuracy and F1-score metrics
6. **Deployment**: Create a Streamlit app for live interaction

## Key Results or Observations
- Successfully fine-tuned BERT on AG News dataset
- Achieved high accuracy and F1-score on test set
- Created interactive Streamlit app for real-time classification
